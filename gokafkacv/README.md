# GoKafkaCV

Automated docker build image containing:

+ FROM adaickalavan/confluent-kafka-go
  + Alpine
  + Golang
  + Librdkafka
  + Confluent-Kafka-Go

+ FROM this-docker-file
  + Golang:latest (Debian based image)
  + GoCV
  + OpenCV 4.0.0

Docker image available at [DockerHub](https://hub.docker.com/u/adaickalavan/).
